# NFHS Health Analysis

## Samsung Innovation Campus Hackathon Project

### Objective
Analyze the NFHS-5 (National Family Health Survey) dataset to identify health patterns across Indian states.

---

## Dataset
- NFHS-5 States Dataset

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Folium

---

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Matrix
- Bar Charts
- Scatter Plot
- Interactive India Choropleth Map
- Health Indicator Visualization

---

## Files

- phases.py → Complete project code
- NFHS-5-States.csv → Dataset
- india_state.geojson → India state boundaries
- india_health_map.html → Interactive map
